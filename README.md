# tally

#### 介绍
牌局记账助手是一款专为打牌爱好者设计的开源项目，旨在帮助用户轻松管理打牌时的花销和统计数据。
通过结合Java后台和微信小程序前端，提供了一套完整的打牌记账解决方案，用户可以随时随地记录各种牌局的开销和胜负情况。
